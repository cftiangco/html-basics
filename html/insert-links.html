The HTML a tag, or anchor tag, is used to create a hyperlink or link to another web page or resource.
It requires the href attribute to specify the destination URL, allowing users to click on the link and navigate to the linked page or resource. 

<a href="https://www.youtube.com/">Go To Youtube!</a>

Full Source Code

<!DOCTYPE html>
<html>
<body>
    <p><a href="https://www.youtube.com/">Go To Youtube!</a></p>
</body>
</html>

Output

Go To Youtube!
