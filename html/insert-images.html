HTML Images

The HTML img tag is used to embed an image into a web page.
It requires the src attribute to specify the image URL. 

<img src="apples.jpg" alt="apple images" />

Full Source Code

<!DOCTYPE html>
<html>
<body>
                
<h2>HTML Image</h2>
<img src="apples.jpg" alt="Girl in a jacket" width="500" height="600">
                
</body>
</html>
